# Changelog — NiTe v1.0.0

Все значимые изменения этого проекта задокументированы в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.1.0/),
и проект придерживается [Semantic Versioning](https://semver.org/lang/ru/).

## [1.0.0] — 2025-02-20

### Добавлено

#### Ядро приложения
- **Управление задачами** — полнофункциональная система создания, редактирования и удаления задач с приоритетами, сроками и тегами
- **Система тегов** — создание и управление цветными тегами для категоризации задач
- **Сценарии** — шаблоны задач для быстрого создания типовых наборов задач (утро, рабочий день, отпуск и т.д.)

#### Питание
- **Библиотека продуктов** — база данных продуктов с информацией о калориях, белках, жирах и углеводах
- **Дневник питания** — отслеживание приёмов пищи в течение дня

#### AI и отчёты
- **AI-анализ** — генерация отчётов о продуктивности с использованием Mistral AI
- **Рекомендации** — персонализированные советы по улучшению рабочего процесса

#### Уведомления
- **Локальные уведомления** — напоминания о задачах с гибкой настройкой времени
- **Поддержка часовых поясов** — корректная работа с разными часовыми поясами

#### Виджеты
- **Home Screen Widget** — быстрый доступ к задачам с главного экрана Android

#### Интерфейс
- **Адаптивный дизайн** — поддержка мобильных и десктопных устройств
- **Тёмная/светлая тема** — переключение между темами оформления
- **Боковая навигация** — удобная навигация по разделам приложения

#### Локализация
- **Мультиязычность** — поддержка русского и английского языков (flutter_localizations)

### Зависимости

- `get: ^4.6.6` — управление состоянием
- `hive: ^2.2.3` + `hive_flutter: ^1.1.0` — локальная база данных
- `flutter_local_notifications: ^17.2.4` — локальные уведомления
- `image_picker: ^1.1.2` — выбор изображений
- `dio: ^5.7.0` — HTTP-клиент
- `intl: ^0.20.2` — форматирование дат и времени
- `uuid: ^4.4.2` — генерация уникальных идентификаторов
- `path_provider: ^2.1.4` — доступ к файловой системе
- `url_launcher: ^6.3.1` — открытие ссылок
- `home_widget: ^0.7.0` — домашний виджет
- `timezone: ^0.9.4` — поддержка часовых поясов

### Техническая информация

- **Версия Flutter**: 3.10.8+
- **Версия Dart**: 3.10.8+
- **Платформы**: Android
- **Архитектура**: Clean Architecture с разделением на data, domain и presentation слои
- **Паттерн**: MVC с использованием GetX

### Известные ограничения

- AI-отчёты требуют подключения к интернету
- Home Widget доступен только на Android
- Некоторые функции могут отличаться в зависимости от платформы
