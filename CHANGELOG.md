## [1.2.0] — 2026-02-21

### Добавлено

#### Должник (Overdue Tasks)
- **Экран «Должник»** — Tinder-стиль карточки просроченных задач при запуске
- Свайп вправо → перенести на сегодня (автоматически подбирается свободное время)
- Свайп влево → выбор новой даты через DatePicker
- Свайп вверх → пометить как выполненную (архивировать)
- Свайп вниз → оставить на исходном дне
- Кнопки-действия под карточкой как альтернатива свайпам
- Прогресс-бар и счётчик оставшихся карточек
- Кнопка «Всё сегодня» — перенести все просрочки одним нажатием
- Цветовая индикация просрочки (красный — критично, жёлтый — можно перенести)
- **AI-подсказки** — анализ просрочек и советы по группировке задач
- Автозапуск при старте приложения если есть просроченные задачи
- Доступен из бокового меню
- Настройки: тумблер включения экрана и тумблер AI-подсказок

#### Экспорт / Импорт данных
- **Экспорт в JSON** — полный бэкап задач, тегов, продуктов питания и сценариев
- **Импорт из JSON** — восстановление данных из файла (дубликаты пропускаются)
- Доступно в Настройках → «Данные»
- Экспорт открывает системный шаринг (мессенджеры, облако, почта)

#### Push-уведомления с действиями (Android)
- Кнопка **«✅ Выполнить»** — отмечает задачу выполненной прямо из шторки
- Кнопка **«⏰ +15 мин»** — откладывает напоминание на 15 минут
- Payload с taskId для идентификации задачи в фоне

#### Предпросмотр задачи
- **Экран предпросмотра** — нажатие на задачу открывает режим просмотра, не редактирования
- Кнопка «Редактировать» внизу для перехода в форму редактирования
- Кнопка ✓/○ в AppBar для быстрого переключения выполнения

#### План питания
- **Экран плана питания** — недельный план с блоками по приёмам пищи
- 4 блока: 🌅 Завтрак / ☀️ Обед / 🌙 Ужин / 🍎 Перекус
- Прогресс-бары КБЖУ (калории, белки, жиры, углеводы)
- Добавление продуктов из библиотеки, редактирование граммовки
- **Нормы КБЖУ** — синхронизированы между планом питания и настройками
- Кнопка «Сохранить как сценарий» — создаёт шаблон недели с задачами

#### Памятка (Help)
- Быстрая навигация по разделам (кнопки-теги вверху)
- Сворачиваемые разделы с иконками
- Актуальный контент по всем функциям приложения

#### Настройки AI — улучшенный флоу
- 4 шага: Провайдер → API ключ → Проверка соединения → Модель
- Кнопки «← Назад» на каждом шаге
- Отдельный шаг проверки соединения с индикатором статуса
- Раздел «Нормы питания» в настройках

### Исправлено

- **Уведомления** — добавлены разрешения `USE_EXACT_ALARM` и `SCHEDULE_EXACT_ALARM` для Android 12+
- **Ежедневный отчёт** — теперь генерируется автоматически в 22:00, а не только по запросу
- **Еженедельная ретроспектива** — исправлен баг с устаревшим ключом Mistral при использовании других провайдеров
- **Android виджет** — `registerBackgroundCallback` → `registerInteractivityCallback`
- **Deprecated API** — `withOpacity` → `withValues(alpha:)` по всему коду

### Зависимости (новые)

- `share_plus: ^10.1.3` — шаринг файлов экспорта
- `file_picker: ^8.1.6` — выбор файла для импорта

## [1.1.0] — 2026-01-15

### Добавлено

#### AI-провайдеры
- **Мультипровайдерная поддержка** — Mistral, OpenAI, Gemini, DeepSeek, Qwen, Claude, Groq
- Универсальный `AiService` через OpenAI-совместимый API
- Выбор провайдера и модели в настройках
- Тестирование соединения прямо в настройках

#### Статистика и отчёты
- **Экран статистики** — КБЖУ за день/неделю, учёт веса, профиль тела
- **Экран отчётов** — история AI-отчётов (ежедневные и еженедельные)
- **Ежедневный отчёт** — итоги дня в 22:00 с советом на завтра
- **Еженедельная ретроспектива** — каждый понедельник в 12:00

#### Интерфейс
- **Режим «День»** — просмотр задач одного дня с навигацией
- **Фильтр по тегам** — горизонтальная строка фильтров на главном экране
- **Профили** — быстрое переключение между наборами задач по тегу
- Настройка стороны бокового меню (слева / справа)

#### Задачи
- **Подзадачи** — чекбоксы внутри задачи с drag-and-drop сортировкой
- **Привязка нескольких продуктов** к одной задаче с тегом «Еда»
- Напоминания о задачах за 5/10/15/30 минут до начала

## [1.0.0] — 2025-12-01

### Добавлено

#### Ядро приложения
- **Управление задачами** — создание, редактирование, удаление задач с приоритетами (0–5), сроками и тегами
- **Система тегов** — цветные теги с эмодзи для категоризации задач
- **Сценарии** — шаблоны задач на неделю для быстрого планирования
- **Drag-and-drop** — перетаскивание задач между днями на главном экране

#### Питание
- **Библиотека продуктов** — карточки с КБЖУ, фото и описанием
- **Привязка к задачам** — задачи с тегом «Еда» связываются с карточками продуктов

#### AI
- **Оценка приоритета через AI** — Mistral AI рекомендует приоритет задачи

#### Уведомления и виджет
- **Локальные уведомления** — напоминания о задачах и отчёты
- **Home Screen Widget** (Android) — ближайшая задача на главном экране

#### Интерфейс
- Чёрно-серая тема (Neomorphism/Glassmorphism)
- Еженедельный планировщик с горизонтальной прокруткой
- Боковое меню с навигацией по разделам

- `hive: ^2.2.3` + `hive_flutter: ^1.1.0` — локальная БД

- `image_picker: ^1.1.2` — выбор фото
- `dio: ^5.7.0` — HTTP-клиент
- `intl: ^0.20.2` — форматирование дат
- `uuid: ^4.4.2` — уникальные ID
- `path_provider: ^2.1.4` — файловая система
- `url_launcher: ^6.3.1` — открытие ссылок
- `home_widget: ^0.7.0` — домашний виджет
- `timezone: ^0.9.4` — часовые пояса

### Техническая информация

- **Flutter**: 3.10.8+
