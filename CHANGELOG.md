# Changelog — NiTe

Все значимые изменения этого проекта задокументированы в этом файле.

---

## [1.2.2] — 2026-02-22

### Сценарии
- Добавлена привязка **продуктов питания** к задачам сценария — секция еды появляется в диалоге редактирования задачи, если выбран тег «Еда»
- Добавлены **подзадачи** в шаблоны задач сценария — теперь хранятся в модели и передаются при применении сценария
- Исправлена конвертация `weekday` в `createScenarioFromAiPlan` — убрано ошибочное смещение `-1`
- При применении сценария (`applyScenario`) подзадачи и продукты питания корректно переносятся в созданные задачи

### План питания
- «Сохранить как сценарий» теперь передаёт `foodItemIds` и `foodItemGrams` в задачи сценария вместо текстового описания

### Исправления
- `TaskDetailScreen`: КБЖУ каждого продукта теперь рассчитывается по индивидуальному граммажу (`foodItemGrams[id]`) вместо устаревшего общего поля `foodGrams`

---

## [1.2.1] — 2026-02-21

### Уведомления
- Заменён `flutter_local_notifications` на `awesome_notifications` — исправлена ошибка "Missing type parameter", из-за которой напоминания не отправлялись
- Уведомления теперь надёжно приходят когда приложение активно, свёрнуто и закрыто
- Кнопка «✅ Выполнить» заменена на «📋 Открыть» — открывает задачу прямо из уведомления
- Исправлен звук уведомления (теперь системный, а не будильник)
- По умолчанию уведомления включены для новых пользователей

### Настройки
- Добавлен выбор времени ежедневного отчёта (18:00–23:00)

### Отчёты
- Полностью переработан экран отчётов: анимированный hero-блок, бар-чарт по дням недели, donut-чарт по тегам, прогресс-бары выполнения
- Автогенерация отчётов через WorkManager даже при закрытом приложении

### Библиотека продуктов
- Мультивыбор: long press → режим выделения с галочками
- Действия над выбранными: **Скрыть** / **Восстановить** / **Удалить** (с подтверждением)
- Скрытые продукты не отображаются в плане питания и при привязке к задаче с тегом «Еда»
- Кнопка 👁️ в AppBar для просмотра скрытых продуктов

## [1.2.0] — 2026-02-21

### Добавлено

- **Экран «Должник»** — Tinder-стиль карточки просроченных задач при запуске
- **Экспорт / Импорт данных** — полный бэкап в JSON
- **Push-уведомления с действиями** — «Выполнить» и «+15 мин» прямо из шторки
- **Экран предпросмотра задачи**
- **План питания** — недельный план с блоками по приёмам пищи и КБЖУ
- **Памятка** — встроенная документация по функциям

## [1.1.0] — 2026-01-15

### Добавлено

- **Мультипровайдерная поддержка AI** — Mistral, OpenAI, Gemini, DeepSeek, Qwen, Claude, Groq
- **Экран статистики** — КБЖУ, учёт веса, профиль тела
- **Экран отчётов** — история AI-отчётов
- **Подзадачи** — чекбоксы внутри задачи
- Напоминания за 5/10/15/30 минут до задачи
