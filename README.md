## ✨ Возможности

### 📅 Еженедельный планировщик
- Просмотр задач по неделям — свайп влево/вправо
- Режимы **«Неделя»** (7 колонок) и **«День»** (один день на весь экран)
- **Drag & Drop** — перетаскивание задач между днями
- Фильтр задач по тегам (только реально присутствующие)

### ✅ Задачи
- Название, описание, теги, время начала/конца, приоритет (0–5)
- Цветная полоска приоритета на карточке
- Подзадачи с чекбоксами (автозакрытие родительской при выполнении всех)
- AI-оценка приоритета по запросу

### 🍽️ Библиотека питания
- Карточки блюд с фото, КБЖУ **на 100г**, описанием
- При добавлении блюда к задаче — указываешь граммы → КБЖУ пересчитывается автоматически
- AI-план питания на неделю (с учётом веса, роста, возраста, пола)

### 🤖 Искусственный интеллект
Поддерживаемые провайдеры:
| Провайдер | Модели |
|---|---|
| Mistral AI | mistral-small, mistral-medium, mistral-large |
| OpenAI | gpt-4o-mini, gpt-4o, gpt-4-turbo |
| Google Gemini | gemini-1.5-flash, gemini-1.5-pro |
| DeepSeek | deepseek-chat, deepseek-reasoner |
| Qwen (Alibaba) | qwen-turbo, qwen-plus, qwen-max |
| Anthropic Claude | claude-3-haiku, claude-3-sonnet, claude-3-opus |
| Groq | llama3-8b, llama3-70b, mixtral-8x7b |

**AI-функции:**
- 🎯 Оценка приоритета задачи
- 📊 Ежедневный отчёт в 22:00 (итоги дня + совет на завтра)
- 📈 Еженедельный отчёт каждый понедельник в 12:00
- 🥗 Генерация плана питания на неделю (≥5 приёмов пищи в день)

### 🎬 Сценарии
- Шаблоны недели для быстрого планирования
- Применение на текущую или следующую неделю
- Задачи привязаны к дню недели (ПН–ВС), а не к дате

### 📊 Статистика
- КБЖУ за день и неделю
- Учёт веса с динамикой
- AI-план набора массы (по формуле TDEE + 300 ккал)

### 📋 Отчёты
- Все AI-отчёты сохраняются локально
- Просмотр по вкладкам: «Ежедневные» / «Еженедельные»
- Уведомление с кратким текстом + развёрнутый отчёт в приложении

### 📲 Виджет домашнего экрана (Android)
- Показывает **ближайшую предстоящую задачу**
- Название, время, дата, эмодзи тега, цветная полоска приоритета
- Обновляется автоматически при изменении задач
- Тап — открывает приложение

## 🚀 Быстрый старт

### Требования
- Flutter SDK >= 3.10
- Dart SDK >= 3.0
- Android SDK (для Android сборки)

### Установка

```bash
# Клонировать репозиторий
git clone https://github.com/Binarer/nite.git
cd nite

# Установить зависимости
flutter pub get

# Сгенерировать Hive адаптеры (если нужно)
dart run build_runner build --delete-conflicting-outputs

# Запустить
flutter run
```

### Настройка AI

1. Открыть приложение → **Настройки**
2. В разделе **ИИ**: выбрать провайдера → вставить API ключ → проверить → выбрать модель
3. Готово! AI-функции активны

> Приложение работает **полностью офлайн**. AI-функции требуют интернета.

├── core/
│   ├── constants/      # AppConstants (typeId, контакты)
│   ├── routes/         # AppRoutes
